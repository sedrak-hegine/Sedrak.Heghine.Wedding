Sedrak.Heghine.Wedding

Wedding invitation website for Sedrak & Heghine.

## Free hosting on GitHub Pages

This repository is configured with a GitHub Actions workflow that builds the site with Vite and deploys it to GitHub Pages automatically on pushes to `work` or `main`.

Expected public URL:

- `https://sedrak-hegine.github.io/Sedrak.Heghine.Wedding/`

## Local development

```bash
npm install
npm run dev
```
